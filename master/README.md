# AiHack. Команда 101. Проект "Протек".

Решаем кейс "Protek" по созданию рейтинговой системы производителей лекарств на основе отзывов, нарушений и данных из открытых источников.

---------------------------------------

# Данные

В папке /data:
1. expert.csv - Экспертная оценка нарушений производителей.
2. prepared_google.csv - Подготовленные данные из google.
3. fed_prepared.csv - Подготовленные данные из федерального источника. 
4. settings.xlsx - Настройки - количество кластеров, веса кластеров.
5. gos_prepared.csv - Подготовка гос данных.
6. Report_Database.xlsx - Изъятия.

---------------------------------------

# Модели

В текущей дериктории:
1. prepare_*.ipynb - Тетрадки для подготовки данных.
2. rating_model.ipynb - Методы для оценки производителей.
3. model.ipynb - Регрессионная модель для предсказаний вероятности наступления следующего нарушения.
4. visualization.ipynb - Визуализцаия.
---------------------------------------

# Бот

В bot.py реализован Телеграмм-бот для общения с системой. В его функции входит показ рейтинга производителей, настройка для кластеризации, поиск по названию лекарства, поиск по производителю.

---------------------------------------

# Результаты

В result.csv лежит список из 250+ производителей, обогащенные внешними и статистическими данными. В столбце "Итоговый рейтинг" - рейтинг производителя от 0 до 1 (чем больше - тем хуже).
